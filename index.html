1. Employee Table & Operators

Creation, Insertion of 10 rows, Comparison, Logical, and Set Operators.

Step 1: Create Table 
CREATE TABLE Employee (
    Name VARCHAR(50),
    Designation VARCHAR(30),
    Gender CHAR(1),
    Age INT,
    DOJ DATE,
    Salary DECIMAL(10,2)
);
Step 2: Insert 10 Rows 
INSERT INTO Employee VALUES 
('Arun', 'Manager', 'M', 35, '2020-01-15', 60000),
('Banu', 'Developer', 'F', 28, '2021-03-10', 45000),
('Charles', 'Analyst', 'M', 30, '2019-06-20', 50000),
('Divya', 'HR', 'F', 26, '2022-08-01', 35000),
('Ezra', 'Tester', 'M', 24, '2023-01-05', 25000),
('Fahad', 'Developer', 'M', 29, '2021-05-15', 48000),

('Indu', 'Analyst', 'F', 27, '2020-09-30', 52000),
('Jack', 'Developer', 'M', 32, '2019-12-01', 55000);
Step 3: Comparison Query (> 40k Salary) 
SELECT * FROM Employee 
WHERE Salary > 40000;
Step 4: Logical Query (Male AND Manager) 
SELECT * FROM Employee 
WHERE Gender = 'M' AND Designation = 'Manager';
Step 5: Set Operator (UNION) 
SELECT Name FROM Employee WHERE Designation = 'Manager'
UNION
SELECT Name FROM Employee WHERE Designation = 'Analyst';
2. TCL Commands (Commit & Rollback)

Performing Transaction Control Statements.

Step 1: Create EMP Table 
CREATE TABLE EMP (
    Empno INT PRIMARY KEY,
    Ename VARCHAR(30),
    Sal DECIMAL(10,2)
);
Step 2: Insert & Commit (Save) 
INSERT INTO EMP VALUES (101, 'Ravi', 25000);
COMMIT;
Step 3: Delete & Rollback (Undo) 
DELETE FROM EMP WHERE Empno = 101;
ROLLBACK;
SELECT * FROM EMP;
3. Single Row Functions

String, Numeric, and Date functions.

Step 1: String Functions (Upper, Length) 
SELECT Name, UPPER(Name), LENGTH(Name) 
FROM Employee;
Step 2: Numeric Function (Round) 
SELECT Name, Salary, ROUND(Salary, -3) 
FROM Employee;
4. Aggregation & Grouping

Using GROUP BY, HAVING, and Aggregate functions.

Step 1: Group By Designation 
SELECT Designation, COUNT(*) AS Count, AVG(Salary) AS Avg_Sal
FROM Employee
GROUP BY Designation;
Step 2: Having Clause (Filter Groups) 
SELECT Designation, COUNT(*) 
FROM Employee
GROUP BY Designation
HAVING COUNT(*) > 1;
5. Bus Reservation System

Creating tables with Constraints.

Step 1: Create Bus Table (Check Constraint) 
CREATE TABLE Bus_Details (
    Bus_ID INT PRIMARY KEY,
    Type VARCHAR(10) CHECK (Type IN ('AC', 'NON-AC'))
);
Step 2: Create Passenger Table (Foreign Key) 
CREATE TABLE Passenger (
    Ticket_ID INT PRIMARY KEY,
    P_Name VARCHAR(50),
    Bus_ID INT,
    FOREIGN KEY (Bus_ID) REFERENCES Bus_Details(Bus_ID)
);
Step 3: View Constraints 
SELECT constraint_name, constraint_type 
FROM user_constraints 
WHERE table_name = 'BUS_DETAILS';
