2. Transaction Control (TCL)

Commit & Rollback
1. Create EMP Table 
CREATE TABLE EMP (
    Empno NUMBER PRIMARY KEY,
    Ename VARCHAR2(10),
    Sal NUMBER(7,2)
);
2. Insert & Commit 
INSERT INTO EMP VALUES(1, 'Ravi', 5000);
COMMIT;
3. Insert, Rollback & View 
INSERT INTO EMP VALUES(2, 'Kumar', 6000);
ROLLBACK;

SELECT * FROM EMP;

3. Single Row Functions

String, Date, Numeric
a) Current Date 
SELECT SYSDATE FROM dual;
b) Round Salary (+15%) 
SELECT Ename, Sal, ROUND(Sal * 1.15) AS New_Sal 
FROM EMP;
c) Initcap & Length 
SELECT INITCAP(Ename), LENGTH(Ename) FROM EMP;
d) Months Worked 
SELECT Ename, TRUNC(MONTHS_BETWEEN(SYSDATE, Hiredate)) 
AS Exp_Months FROM EMP;

4. Aggregate Functions

Group By & Having
1. Min, Max, Avg, Sum 
SELECT MIN(Sal), MAX(Sal), AVG(Sal), SUM(Sal) FROM EMP;
2. Group By Job 
SELECT Job, COUNT(*) FROM EMP GROUP BY Job;
3. Having Clause (Count > 1) 
SELECT Deptno, COUNT(*) FROM EMP 
GROUP BY Deptno 
HAVING COUNT(*) > 1;

5. Bus Reservation System

Constraints (PK/FK)
1. Create Bus Table 
CREATE TABLE Bus (
    Routeno NUMBER PRIMARY KEY,
    Source VARCHAR2(20),
    Destination VARCHAR2(20)
);
2. Create Passenger Table 
CREATE TABLE Passenger (
    Pid NUMBER PRIMARY KEY,
    Pname VARCHAR2(20),
    Gender VARCHAR2(6)
);
3. Create Booking Table 
CREATE TABLE Booking (
    Pid NUMBER REFERENCES Passenger(Pid),
    Routeno NUMBER REFERENCES Bus(Routeno),
    JDate DATE,
    PRIMARY KEY(Pid, Routeno)
);

6. Implementation of Joins

Inner, Outer, Self
1. Inner Join 
SELECT S.Sname, B.Bname 
FROM Sailors S 
INNER JOIN Reserves R ON S.Sid = R.Sid
INNER JOIN Boats B ON R.Bid = B.Bid;
2. Left Join 
SELECT S.Sname, R.Day 
FROM Sailors S 
LEFT JOIN Reserves R ON S.Sid = R.Sid;
3. Right Join 
SELECT S.Sname, B.Bname 
FROM Sailors S 
RIGHT JOIN Reserves R ON S.Sid = R.Sid
RIGHT JOIN Boats B ON R.Bid = B.Bid;
